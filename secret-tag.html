Upcoming Game: <TAG>

<TAG> is a fast-paced multiplayer tag game on Roblox where players compete across large arena-style maps using special gear that changes each round.

Core Gameplay

The game is built around movement, chasing, and escaping. Players avoid being tagged while navigating maps with multiple routes, platforms, and obstacles that create dynamic chase paths.

Round Gear

During rounds, players can obtain classic Roblox-style gear, including:

  • Bloxy Cola for speed boosts
  • Rainbow Carpet for aerial mobility
  • Sword for close combat interactions

Lobby And Replayability

A fully custom lobby includes an interactive gear showcase where players can view and cycle through featured items before matches. Different map and gear combinations make rounds feel fresh and replayable.

This project was primarily developed by me, including gameplay systems, mechanics, UI, and core design, with additional help from builders for map environments.
